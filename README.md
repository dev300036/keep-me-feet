# Keep Me Feet - Backend API

## Overview

Keep Me Feet is a nutrition and health management platform that connects users with nutritionists and provides personalized diet planning services.

This repository contains the backend developed using PHP and MySQL, providing APIs, authentication, user management, diet planning, nutritionist management, and payment-related functionalities.

---

* CSS

* PHPMailer

## Installation

1. Install XAMPP.
2. Copy the project folder into:

```text
C:\xampp\htdocs\
```

3. Start:

   * Apache
   * MySQL

4. Create a MySQL database.

5. Import the SQL file.

6. Configure database connection settings.

7. Open in browser:

```text
http://localhost/client_project
```
